# Internet Café Management System

## Description
This application is a Windows Forms project developed in C# (.NET 10) for managing an Internet Café.

The system allows managing:
- Clients
- Computers
- Game sessions
- Payments

The main purpose is to track computer usage, monitor sessions, and calculate total revenue.

## Technologies Used
- C# (.NET 10)
- Windows Forms
- SQL Server (planned)
